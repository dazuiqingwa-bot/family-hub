# skill: hehe-learning
# 何时加载：老板发来的消息涉及禾禾的具体错题时

## 触发条件（三个同时满足）

条件 A：出现具体学科
数学 / 语文 / 物理 / 化学 / 生物 / 英语 中的任意一个

条件 B：出现错误信息
- 错了 / 答错 / 做错 / 没做对 / 扣分 / 得0分
- 她的答案是…… / 她写的是……
- 不会 / 不理解 / 搞不清楚

条件 C：有题目相关内容
题目描述 / 知识点 / 题型等信息

## 执行流程

1. 从消息中提取：学科、题目描述、禾禾的答案、正确答案
2. 信息不完整 → 先反问老板补齐，再继续
3. 按 Learning Center 工作流执行：错因分析 + 针对性练习生成
4. 调用 Notion skill 写入禾禾错题总库
5. 回复老板写入确认（题目标题 + 页面链接）

信息补齐后直接写入，不需要再次确认。禁止二次确认。

## Notion 参数

- 数据库 ID：`329d76be33b0804e9db3d9ee9fde3429`
- API key 路径：`~/.config/notion/api_key`
- 脚本路径：`~/.openclaw/workspace/scripts/notion-insert.py`
- 年级默认值：初二下
- 录入日期：自动取当天

## 不触发的情况

- "禾禾数学考了80分" → 无错题信息
- "禾禾数学最近有点落后" → 无具体题目
- "这道数学题怎么做？" → 无禾禾做错的信息，ASA 直接解题
